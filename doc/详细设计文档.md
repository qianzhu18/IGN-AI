# 一、为什么你现在这版会有“不清晰”的感觉

这个不只是模糊一个问题。

核心是 5 个因素叠加：

### 1. 卡片的失焦太重

你现在用“模糊”来区分焦点和非焦点。
这在概念图里成立，在 production 成品里很容易显得像：

* 还没加载完
* 还在调试
* 视觉故意做脏了

更适合 production 的做法是：

* 焦点卡更亮
* 非焦点卡更暗
* 但依旧保持可读

### 2. 中文大字虽然猛，但字体渲染不够稳

“不是围观，而是上场”这一类大字，现在冲击是有的。
问题是：

* 字体太轻
* 斜势大
* 笔画细
* 在深背景和 glow 叠加下容易发虚

### 3. 背景雾层偏厚

你现在有一层“深色 + 暖色雾 + 蓝紫雾”的背景逻辑。
但它对文字清晰度有副作用：

* 对比不稳定
* 某些区块像字浮在雾里

### 4. 卡片底和页面底太接近

有些卡片虽然加了描边，但底色仍然和背景太近。
结果会是：

* 卡片轮廓不够干净
* 卡片像漂浮的一块黑片
* 不够“硬朗”

### 5. 你用了“高级感的语言”，但还没做到“完成态的秩序”

也就是：

* 氛围到了
* 但界面还不像最终精修成品

# 二、颜色为什么不够贴合 IGNAI

这块是你说得最准的地方。

你现在的色彩更偏：

* 深夜
* 冷静
* 紫蓝宇宙感
* 少量橙光

这套配色可以做出“未来感”，
但很难自然地表达 IGNAI 最重要的那层气质：

**点燃、煽动、让人上场。**

所以我给你的新判断是：

## IGNAI 的颜色主轴不该是“紫 + 橙”

而应该是：

**深夜黑蓝 + 火种橙金 + 电蓝信号 + 微量紫色过渡**

重点在这里：

### 橙色要变成真正主角

不是现在这种“局部暖色点缀”，
而是让用户一进来就感觉到：

**这地方是热的。**

### 紫色要退为辅助

紫色可以保留空间感和科技感，
但不能再跟橙色平分品牌主导权。

因为紫色一多，就容易往：

* 梦幻
* 宇宙
* 抽象艺术
  那边滑。

IGNAI 要的是：

* 火种
* 信号
* 行动
* 社群温度

# 五、IGNAI 的图像系统应该怎么建

我在文档里给你重写成了 3 类图像语言：

## 1. Ignite 火种意象

用来表达：

* 点燃
* 触发
* 热源
* 从暗场里亮起来

适合：

* 粒子火花
* 局部发亮
* 一束能量被点亮
* 一群灯逐步亮起

---

## 2. Signal 信号意象

用来表达：

* 被看见
* 节点连接
* 社群传播
* 连接本地和世界

适合：

* 光点连线
* 城市信号
* 数据脉冲
* 流线轨迹

---

## 3. Human Energy 人的热度

这是你现在最缺的那层。

IGNAI 不是纯技术站，
它一定要有人和人的热度。

适合：

* 夜间共创场景
* 局部聚会剪影
* 围桌交流
* 分享场、灯光、侧影、手势、桌面、屏幕反光

不是做纪实摄影首页，
而是用更抽象但有“现场感”的方式，把人带回来。
