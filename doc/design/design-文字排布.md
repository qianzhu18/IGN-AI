下面是一份可以直接交给设计/前端去改的**统一排布与文字修改建议**。我会把它整理成“统一规则 + 每屏改法 + 推荐文案”，目标是让这个社区网页从视觉概念稿更接近生产级官网。

---

# IGNAI 社区官网排布与文字修改建议

## 一、整体改版目标

当前页面已经有完整的视觉气质：深色、火光、技术感、本地社区感都成立。主要问题集中在文字排布：

1. 中文标题断行过碎，影响阅读。
2. 英文标题、中文标题、卡片标题层级不够稳定。
3. 左右区域对齐规则不统一。
4. 卡片内部排版没有形成统一模板。
5. 每个区块视觉都很强，导致页面节奏略满。
6. 部分文案偏口号，缺少产品官网需要的解释力和转化力。

改版目标是：

```text
建立统一阅读路径：
你是谁 → 为什么存在 → 你相信什么 → 你有什么不同 → 我为什么加入
```

---

# 二、全站统一排布规则

## 1. 页面容器

所有主体内容统一放进同一个容器系统：

```css
max-width: 1200px;
margin: 0 auto;
padding-left: 32px;
padding-right: 32px;
```

移动端可调整为：

```css
padding-left: 20px;
padding-right: 20px;
```

---

## 2. 主体网格

所有左右结构统一使用 12 栏逻辑：

```css
display: grid;
grid-template-columns: 5fr 7fr;
gap: 72px;
align-items: center;
```

建议：

```text
左侧：主叙事区，放 section label / 大标题 / 正文 / CTA
右侧：内容承载区，放图片卡片 / 信息卡片 / 关键词 / 加入渠道
```

这样全站会变得稳定，不会每屏都像重新设计。

---

## 3. 区块间距

统一节奏：

```text
Hero 高度：min-height 760px
普通 Section 上下 padding：120px
紧凑 Section 上下 padding：96px
区块内部大标题到正文：24px
正文到卡片/按钮：32px
卡片之间 gap：20px 或 24px
```

---

## 4. 文字层级

建议固定 5 层：

### 英文 Display Title

用于品牌情绪标题。

```css
font-size: clamp(56px, 7vw, 112px);
line-height: 0.95;
letter-spacing: -0.04em;
```

使用位置：

```text
Ignite before AGI.
What is IGNAI?
Ignite people before hype.
What makes IGNAI different
Join the fire.
```

---

### 中文 Section Title

用于解释核心信息。

```css
font-size: clamp(36px, 4vw, 64px);
line-height: 1.12;
font-weight: 800;
letter-spacing: -0.03em;
```

注意：中文标题不要拆词。

错误倾向：

```text
有技术密
度，
也有人的温
度。
```

建议：

```text
有技术密度，
也有人的温度。
```

---

### Section 正文

```css
font-size: 16px;
line-height: 1.8;
color: rgba(255,255,255,0.72);
max-width: 520px;
```

---

### 卡片标题

```css
font-size: 22px;
line-height: 1.32;
font-weight: 700;
```

---

### 卡片说明

```css
font-size: 14px;
line-height: 1.7;
color: rgba(255,255,255,0.64);
```

---

## 5. 卡片统一模板

所有卡片统一为：

```text
EYEBROW 小标签
主标题
说明文字
```

示例：

```text
LOCAL ROOTS
本地土壤
从长沙出发，把真实的人聚到一起。
```

卡片规则：

```text
小卡片高度统一
标题最多 2 行
说明最多 2 行
所有卡片左对齐
内边距统一 24px
圆角统一 20px 或 24px
```

```text

卡片：负责结构化信息
```

例如：

```text
Ignite before AGI.
在 AGI 到来之前，先点燃一群真实行动的人。
```

```text
What is IGNAI?

```

```text

把你的表达、行动和信号，带进这团火里。
```

这样中英文就不会互相抢信息层级。

# 八、视觉噪音处理建议

你的背景光效很强，建议做三点控制：

1. 文字背后的光效降低 25% 左右。
2. 中间蓝色纵向光柱不要穿过正文区域。
3. 每个左侧文字区域可以加一层很轻的暗色渐变遮罩。

原则：

```text
背景负责氛围，
不能和文字抢注意力。
```

---

# 九、最优先修改清单

如果你时间有限，先改这 6 件事：

```text
1. 所有中文标题不拆词
2. 所有区块统一左 5 / 右 7 网格
3. 所有卡片统一 eyebrow / title / desc
4. What 区从关键词墙改成社区成员组成
5. Why Now 区从口号改成问题 → 回应
6. Join 区增加加入后能做什么
```

这 6 件改完，页面会立刻更像生产级官网。
