## 已知问题

### 首页

- [ ] Hero 区域 CTA 按钮样式需打磨（扫光效果、hover 状态）
- [ ] 信号卡片（Signal Cards）布局在某些分辨率下可能溢出
- [ ] 成员散点区域 hover 预览卡片位置偶尔偏移
- [ ] 活动区块空状态样式需优化（目前显示"活动筹备中"占位）

### 成员系统

- [ ] 成员目录页搜索/筛选功能待实现（UI-07 后续迭代）
- [ ] 成员详情页社交链接图标在暗色模式下对比度不足
- [ ] 移动端 tap 预览卡片偶尔触发两次

### 导航

- [ ] 移动端汉堡菜单动画在低端设备上卡顿
- [ ] 菜单项 hover 状态在触摸设备上"粘滞"

### 全局

- [ ] 暗色模式下部分文字对比度不达 WCAG AA 标准
- [ ] 滚动性能在大量成员散点时有优化空间
- [ ] 字体加载闪烁（FOIT）在慢网络下明显

### /about 页面

- [ ] 动画入场时机需调整（framer-motion delay 链）
- [ ] 移动端 stats grid 布局挤压

### /events 页面

- [ ] 活动卡片信息密度在窄屏下过高

## 修复建议

1. 优先修复对比度和可访问性问题
2. 其次优化移动端触摸交互
3. 最后打磨动画和视觉细节
